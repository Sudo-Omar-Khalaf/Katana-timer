│ 🛠️ TROUBLESHOOTING                                                      │
└─────────────────────────────────────────────────────────────────────────┘

    Issue: "command not found: katana"
    Fix: $ source ~/.zshrc
         OR use full path: ~/.local/bin/katana

    Issue: Wake-up not working
    Fix: Check BIOS "Wake on RTC" is enabled
         Verify: $ sudo -n rtcwake --version

    Issue: Password prompts for rtcwake
    Fix: Re-run installer: $ ./install-katana.sh

┌─────────────────────────────────────────────────────────────────────────┐
│ 🎯 HOW WAKE-UP WORKS                                                   │
└─────────────────────────────────────────────────────────────────────────┘

    User creates alarm → Katana schedules RTC wake → PC sleeps normally
                      ↓
              RTC wakes PC automatically
                      ↓
                 Alarm rings!

    No user configuration needed!
    No password prompts!
    Works even if PC is fully suspended!
